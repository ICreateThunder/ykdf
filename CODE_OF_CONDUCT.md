# Code of Conduct

## Standards

- Be respectful in disagreement
- Assume good faith
- No harassment, personal attacks, or discrimination
- No sharing private communications without consent
- Acknowledge others' time

## Enforcement

Disputes resolved by maintainers. Report concerns via
[GitHub Private Security Advisories](../../security/advisories/new)
or robert@shalders.co.uk.

Actions: warn, suspend, or ban.
